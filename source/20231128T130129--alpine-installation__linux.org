INSTALLATION GUIDE

- install virtual machine with 1gb and 10gb
- ping google.com for internet connection
- apk add sudo (might not need this, just enable community first)
- sudo apk add htop
- vi /etc/apk/repositories - enable community one
- apk update
- sudo apk add emacs
