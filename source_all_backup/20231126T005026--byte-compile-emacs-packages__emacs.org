[[https://stackoverflow.com/questions/24725778/how-to-rebuild-elpa-packages-after-upgrade-of-emacs][what is it]]

It was useful for me when I [[id:1fb3bb1a-66bb-4baf-a93c-356df7d53954][install 28 emacs on ubuntu]] and then
lost syntax highlighting in js2 mode. I then found [[https://github.com/mooz/js2-mode/issues/72#issuecomment-15176816][this]] fix. did M-:
and wrote this to fix it:

#+begin_src elisp
  ;; dont forget the exact file path
  (byte-recompile-directory (expand-file-name "~/.emacs.d/elpa/js2-mode-20130307.2012/") 0 t)
#+end_src
