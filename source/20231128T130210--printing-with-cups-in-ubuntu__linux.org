- sudo systemctl enable cups
- sudo systemctl start cups
- sudo systemctl status cups
- download drivers from [[https://support.hp.com/lt-en/drivers/selfservice/samsung-xpress-sl-m2022-laser-printer-series/17157279/model/17157283][here]]
- sudo tar -xvzf udl bla folderi ta draiverio
- sudo ./install.sh
- login to cups service - http://localhost:631/
